Overview

Computers are electronic devices that process data and perform tasks using hardware and software. They accept input, process it, store information, and produce output.

  • Input - data entered into the computer through devices like keyboard, mouse, or scanner.
  • Processing - operations performed by the CPU and memory.
  • Storage - saving data on drives, memory, or cloud systems.
  • Output - results shown via monitor, printer, or speakers.

Computer Hardware

Hardware is the physical part of a computer system. It includes the central processing unit, memory, storage, and input/output devices.

  • CPU - the brain of the computer, responsible for executing instructions.
  • RAM - temporary memory used while programs are running.
  • Storage - long-term data storage such as HDDs, SSDs, and flash drives.
  • Motherboard - the main circuit board connecting all components.
  • Peripherals - devices like keyboard, mouse, printer, and monitor.

Computer Software

Software is the set of instructions that tells hardware what to do. It is divided into system software and application software.

  • System software - operating systems and utility programs that manage hardware.
  • Application software - programs used to perform user tasks like word processing, browsing, and spreadsheets.
  • Operating system - examples include Windows, Linux, and macOS.
  • Drivers - software that lets the operating system communicate with hardware devices.

Networks and Internet

A computer network connects multiple devices to share data and resources. The internet is a global network of networks.

  • LAN - Local Area Network, used within a limited area like a home or office.
  • WAN - Wide Area Network, connects devices over larger distances.
  • Protocol - rules that define communication between devices, such as TCP/IP.
  • Router - device that forwards data packets between networks.

Glossary

  • Algorithm - a step-by-step procedure for solving a problem.
  • Binary - a number system with base 2, using 0 and 1.
  • Database - an organized collection of structured information.
  • Firewall - security system that controls incoming and outgoing network traffic.
  • Cloud - internet-based storage and services.
